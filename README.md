# Calculadora de Partidas Rankeadas

Projeto desenvolvido para cálculo de saldo de vitorias e classificação de nivel de jogadores.

## Tecnologias
- JavaScript

## Objetivo
Calcular o saldo de partidas rankeadas (Vitorias - Derrotas) e determinar o nivel do jogador com base no total de vitorias, utilizando funcoes e estruturas condicionais encadeadas.

## Como Executar
1. Clone o repositorio:
   git clone https://github.com/SEU_USUARIO/calculadora-partidas-rankeadas.git

2. Execute o script com o Node.js:
   node index.js
